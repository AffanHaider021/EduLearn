Quick Notes

HTML Cheat Sheet

Common HTML tags you should remember:

  • <h1> to <h6> - Headings
  • <p> - Paragraph
  • <a> - Link
  • <img> - Image
  • <table> - Table
  • <ul> <ol> - Lists
  • <div> - Container

CSS Cheat Sheet

Common CSS properties you should know:

  • color - text color
  • background-color - background
  • font-size - text size
  • margin - outside space
  • padding - inside space
  • border - border around element
  • display - how element is shown

JavaScript Cheat Sheet

Basic JavaScript things to remember:

  • let / const - variables
  • if / else - conditions
  • for / while - loops
  • function - reusable code
  • array - list of values
  • console.log() - print output
  • document.getElementById() - get element

Study Tips

1

Practice writing code every day, even if it is just for 30 minutes.

2

Do not just copy code. Try to understand what each line is doing.

3

When you get an error, read the error message carefully before asking for help.

4

Build small projects. Even a simple calculator or to-do list teaches a lot.

5

Use Google and YouTube. Every programmer uses them all the time.
