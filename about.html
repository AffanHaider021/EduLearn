About EduLearn

A little bit about this website and why it was made

What is EduLearn?

EduLearn is a simple educational website made for students who want to learn different subjects in an easy way. The goal of this website is to provide free learning material for students, especially those who are just starting their studies.

This website was created as a final project for the ICT Lab course at Air University Islamabad. It covers subjects like Computer Science, Mathematics, and Science with simple notes and useful links.

HTML

CSS

CSS is used to style the website. It controls colors, fonts, spacing, and the overall look of every page.

JavaScript

JavaScript is used to add interactivity. Things like the mobile menu, course filter, and quiz are made with JavaScript.
